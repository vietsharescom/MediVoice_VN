## Thuật ngữ Y tế

| Thuật ngữ | Định nghĩa |
|---|---|
| **Mẫu 15/BV-01** | Bệnh án ngoại trú chuẩn VN (TT32/2023) |

| **CCHN** | Chứng chỉ hành nghề khám chữa bệnh |

| **WER** | Word Error Rate — tỷ lệ lỗi từ trong ASR |
| **SOAP** | Subjective/Objective/Assessment/Plan (Canada format) |
| **NER** | Named Entity Recognition — nhận dạng thực thể |

| **PII** | Personally Identifiable Information — thông tin nhận dạng cá nhân |

| **PHI** | Protected Health Information — thông tin sức khỏe được bảo vệ |

## Thuật ngữ Kỹ thuật

| Thuật ngữ | Định nghĩa |
|---|---|
| **Pipeline** | Chuỗi xử lý L0→L10 bất biến (FROZEN) |
| **CCP** | Critical Control Point — điểm kiểm soát quan trọng |
| **FID** | Feature Intent Document — tài liệu ý định tính năng |
| **SRS** | System Requirements Specification — đặc tả yêu cầu hệ thống |
| **RTM** | Requirements Traceability Matrix — ma trận truy vết yêu cầu |
| **Fernet** | Symmetric encryption scheme (cryptography library) |
| **WAL** | Write-Ahead Logging — chế độ ghi trước SQLite |
| **INN** | International Nonproprietary Name — tên thuốc quốc tế |
