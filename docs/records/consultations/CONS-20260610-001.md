## VẤN ĐỀ THỰC TẾ

Cùng thuốc "Amlodipine", 3 bác sĩ nói khác nhau:
  BS Hà Nội:   "am-lô-đi-pin"
  BS Đà Nẵng:  "am-lô-đi-phin"
  BS Sài Gòn:  "am-lo" hoặc "am-lor"

PhoWhisper (trained chủ yếu tiếng Việt chuẩn) còn thêm noise:
  "Amoxicillin" → "a mốc xi lin" (nuốt âm đầu)
  "Metformin"   → "mét pho min" hoặc "me pho min"
  "Paracetamol" → "para xi ta mol" (thêm âm đệm)

Drug Correction Layer cần match "am lo" / "am lor" / "am lô đi phin"
→ tất cả đều là Amlodipine.

## CONSOLIDATED PHONEME RULES (7 rules đồng thuận)

| # | Rule | Ví dụ |
|---|------|-------|
| R1 | Consonant cluster → insert vowel /ơ/ hoặc /a/ | "cl" → "cờ lơ", "str" → "xtơ" |
| R2 | Final -ine/-pine → -pin/-phin (Bắc/Trung) hoặc drop (Nam) | "Amlodipine" → "am-lô-đi-pin" / "am-lo" |
| R3 | /r/: Bắc=/z/, Nam=/ɣ/, Trung=/ɾ/ biến đổi mạnh | "r" trong drug names → varies |
| R4 | Vowel centralization: o↔ô↔ơ, i↔ê (Trung mạnh nhất) | "Metformin" → "mét-pho-min" / "me-pho-miên" |
| R5 | /f/ → /ph/ (Vietnamese phonotactics) | "Fluticasone" → "phlu-ti-ca-sôn" |
| R6 | Tone neutralization fast speech (đặc biệt Nam + Trung) | Multi-syllable drug → flat/shortened |
| R7 | Southern abbreviation: nuốt 2-3 âm cuối | "Amlodipine" → "am-lo" / "am-lor" |

## CENTRAL DIALECT (Đà Nẵng/Quảng Nam) — Consolidated

  Vowels:  ă→e, ô→o, ê→e, ơ→o  (ví dụ: đi thẳng → đi thẻng)
  Tones:   5 tones (ngã/hỏi/nặng merge — không phân biệt rõ)
  Consonants: r/d/gi collapse, /v/→/w/, /tr/→/ch/
  Particles: hỉ, mô, răng, rứa, bữa ni (không phải drug names → cần filter)
  Key insight (ChatGPT): NOT just phoneme — also lexical + discourse noise problem
  Drug examples:
